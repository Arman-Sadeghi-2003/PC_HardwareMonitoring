Tabs:
	1. Home
		Overview Dashboard – real-time summary of all key stats with live charts.
	
		Features:

			🔥 CPU Temperature, Usage (%), Clock speed

			🚀 GPU Temperature, Load (%), Memory usage

			📈 Live charts:

			CPU usage (% over time)

			GPU usage (% over time)

			RAM usage (MB over time)

			🎯 Pie charts:

			RAM: Used vs Available

			GPU memory usage

	2. CPU
		In-depth look at CPU metrics.

		Details:
			Temperature (with min/max)

			Current & base clock speed

			Core vs thread count

			Per-core usage (%) (with small inline charts)

			Power draw (if available)

			Live chart: Usage % over time

			Option to show CPU model

	3. GPU
		Detailed GPU monitoring.

		Details:

			Temperature (min/max)

			GPU Load %

			GPU Memory usage (used vs total)

			Fan speed (if available)

			Live chart: GPU usage or temperature over time

			GPU model info (NVIDIA/AMD/Intel)

	4. RAM
		Memory usage and trends.

		Details:

			Used vs total RAM

			Free memory (in MB or GB)

			Page file usage (if available)

			Live chart: RAM usage trend

			RAM type (DDR3/DDR4/DDR5), speed, slots used

	5. Motherboard
		System sensors (limited by hardware support).

		Details:
			Motherboard temperature (if supported)

			VRM or chipset temperatures

			Fan speeds (CPU/motherboard fans)

			BIOS version (optional)

			Manufacturer info
